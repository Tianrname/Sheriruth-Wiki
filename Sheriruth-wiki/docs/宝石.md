宝石是提升你的战斗力的一个方式，但是你必须要先前往 区域 3 找到 NPC <span style="color: #FFA500;">璃珀</span> 进行解锁

解锁的前置条件是 通关 <span style="color:#00FFFF;">Glimmer</span> 难度下的 Dungeon <span style="color:#FF55FF;">默蚀洞窟</span>，如果你未满足条件就去与 NPC 对话，就会被拒绝

![尚未满足宝石.png](https://youke1.picui.cn/s1/2025/11/19/691db670c3bc7.png)

此 NPC 的位置在 区域 3 的 迴音聚落 (坐标在: X.29 Y.101 Z.129)

达成条件后，与 NPC 对话即可解锁 宝石 功能

之后你便可以在 个人背包界面看到之前的屏障发生了变化

![解锁宝石.png](https://youke1.picui.cn/s1/2025/11/19/691db780d0aa1.png)

## 镶嵌宝石 {style="color: #666666;"}

每个部位的宝石槽都是独立的，且目前固定拥有 3 个 宝石槽，分别会在部位强化等级达到 20 50 80 级的时候解锁

    你只需要进入镶嵌界面，左键点击背包中的宝石，即可自动镶嵌进空闲的槽位中
    若要取下宝石，只需要右键界面中已经镶嵌好的宝石即可 （取下宝石不会有任何亏损）
